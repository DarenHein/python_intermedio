*Proyecto: Administrador de Tareas*
*Descripción:*
Crea un administrador de tareas básico en línea de comandos utilizando Python. Este programa permitirá al usuario agregar, ver y marcar tareas como completadas. Asegúrate de seguir estos parámetros:

*Características:*
1. *Agregar Tareas:* Permite al usuario agregar nuevas tareas con una descripción y una fecha opcional.
2. *Ver Tareas:* Muestra una lista de todas las tareas pendientes con sus descripciones y fechas (si están disponibles).
3. *Marcar Tareas como Completadas:* Permite al usuario marcar tareas como completadas y eliminarlas de la lista de tareas pendientes.
4. *Guardar en un Archivo:* Al finalizar la sesión, guarda las tareas en un archivo para que persistan entre ejecuciones.

*Requisitos:*
- Utiliza funciones para organizar el código y hacerlo modular.
- Utiliza una estructura de datos adecuada para almacenar las tareas (por ejemplo, una lista de diccionarios).
- Implementa manejo de excepciones para evitar errores durante la ejecución.
- Crea una interfaz de usuario sencilla en la línea de comandos para interactuar con el programa.

*Ampliación (Opcional):*
- Agrega la posibilidad de editar tareas existentes.
- Implementa una función de búsqueda para encontrar tareas específicas.
- Crea una versión con interfaz gráfica utilizando una biblioteca como tkinter.

¡Este proyecto te permitirá practicar muchas de las habilidades esenciales de Python, como el manejo de datos, funciones, manejo de excepciones y más!
